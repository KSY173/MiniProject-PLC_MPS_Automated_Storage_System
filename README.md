## 프로젝트 개요

이 프로젝트는 Mitsubishi PLC와 MPS 장비를 이용하여 소재를 금속/비금속으로 판별하고, 판별 결과와 작업 수량에 따라 자동으로 창고에 적재하거나 배출하는 자동화 시스템입니다.

소재 공급, 가공, 이송, 판별, 흡착, 적재, 배출까지의 전체 공정을 PLC 래더 로직으로 제어했습니다.  
또한 서보모터를 이용해 창고 높이에 맞는 정밀 위치 제어를 수행하고, HMI를 통해 장비 상태 모니터링, 작업 선택, 파라미터 변경, JOG 조작, Teaching 기능을 사용할 수 있도록 구현했습니다.

---

## 프로젝트 목표

이 프로젝트의 주요 목표는 다음과 같습니다.

- 소재를 금속 / 비금속으로 판별
- 소재 종류와 수량에 따라 자동으로 창고 위치 결정
- QD77MS2 위치 결정 모듈을 이용한 서보모터 위치 제어
- 전원 공급 후 자동 원점 복귀 구현
- HMI 기반 실시간 모니터링 및 작업 제어
- 수동 JOG 제어 및 서보 위치 Teaching 기능 구현
- 동작 점검, 자동 작업, 선택 적재, 전체 비움, 선택 비움 기능 구현
- 비상 정지 및 비상 해제 로직 구현
- 작업 시간 확인을 위한 Tact Time 측정 기능 구현
- 자동화 장비의 안전성, 편의성, 유지보수성 향상

## 주요 기능

## 1. 전원 공급 후 자동 원점 복귀 및 동작 가능 조건

전원이 공급되면 서보모터가 자동으로 원점 복귀를 수행합니다.

원점 복귀가 완료되면 다음 동작이 수행됩니다.

- `M0` ON
- HMI의 원점 확립 램프 ON
- 전체 실린더 후진
- 장비가 안전한 초기 상태로 전환

예를 들어 공급 실린더 후진 출력 `Y21`은 `M0`의 B접점을 이용해 제어했습니다.  
`M0`가 ON되기 전까지는 공급 실린더 후진 신호를 유지하고, 원점이 확립되면 후진 신호가 OFF되도록 구성했습니다.

이를 통해 장비가 전원 투입 직후 알 수 없는 위치에서 동작을 시작하지 않도록 했습니다.

![전원 공급 후 자동 원점 복귀 로직](./imgs/power.png)

### 동작 가능 조건

시스템은 다음 조건을 모두 만족할 때만 작업을 수행할 수 있습니다.

- 서보모터 원점 확립 완료
- 전체 실린더 후진 완료
- 소재 공급 감지
- 비상 상태가 아님

![원점 확립 및 동작 가능 램프](./imgs/powerlamp.png)

위 조건을 만족하면 HMI의 동작 가능 램프 `M17`이 ON됩니다.

## 2. 동작 점검 모드

동작 점검 모드는 실린더, 드릴 모터, 컨베이어 모터의 동작 상태를 순차적으로 확인하는 기능입니다.

![동작 점검 모드 로직](./imgs/test.png)

동작 점검 버튼 `X15`를 누르면 점검 시퀀스가 시작됩니다.

### 점검 순서

1. 공급 실린더 전진 / 후진
2. 가공 실린더 하강 / 상승
3. 드릴 모터 ON / OFF
4. 분배 실린더 전진 / 후진
5. 컨베이어 모터 ON / OFF
6. 배출 실린더 전진 / 후진
7. 스토퍼 실린더 전진 / 후진
8. 흡착 실린더 전진 / 후진
9. 창고 실린더 전진 / 후진

각 단계에는 완료 센서 조건을 포함했습니다.  
따라서 현재 동작이 완전히 끝나지 않으면 다음 동작으로 넘어가지 않습니다.

점검이 완료되면 동작 점검 완료 램프 `M16`이 ON됩니다.

## 3. 자동 작업 모드

자동 작업 모드는 소재 공급부터 적재 또는 배출까지의 전체 시퀀스를 실행하는 기능입니다.

작업 시작 버튼 `X16`을 누르면 자동 작업이 시작됩니다.

![자동 작업 전체 시퀀스](./imgs/start_flow.png)

### 전체 자동 작업 영상

[전체 자동 작업 영상 보기](https://drive.google.com/file/d/1w_E2pjcpxAljvRO91MlLiYkNVass84lW/view?usp=sharing)

### 자동 작업 순서

1. 공급 실린더가 소재를 공급
2. 가공 실린더 하강
3. 드릴 모터 ON
4. 설정된 가공 시간 동안 가공 수행
5. 설정된 반복 횟수만큼 가공 반복
6. 가공 실린더 상승 + 드릴 모터 OFF
7. 분배 실린더가 소재를 컨베이어로 이동
8. 컨베이어 동작
9. 금속 / 비금속 판별
10. 스토퍼 실린더가 픽업 위치에서 소재 정지
11. 서보모터가 소재 픽업 위치로 이동
12. 흡착 모듈 ON
13. 서보모터가 목표 창고 높이로 이동
14. 흡착 실린더 전진
15. 소재를 놓기 위해 서보모터가 약간 하강
16. 흡착 모듈 OFF
17. 서보모터가 약간 상승
18. 흡착 실린더 후진
19. 다음 작업 대기 위치 또는 원점으로 복귀

가공 시간과 반복 횟수는 HMI에서 변경할 수 있도록 구현했습니다.

## 4. 소재 판별 및 적재 로직

소재는 다음 두 종류로 판별됩니다.

- 금속
- 비금속

각 소재 종류는 별도의 카운터로 수량을 관리합니다.


### 소재 판별 및 창고 적재 위치

![소재 판별 로직](./imgs/work_flow.png)

| 소재 종류 | 카운터 |
|---|---|
| 금속 | `C50` |
| 비금속 | `C51` |


### C50, C51 카운터 로직

![C50, C51 카운터 동작 로직](./imgs/work2.png)

![금속 / 비금속 소재 카운트 화면](./imgs/material_count.png)

![창고 적재 위치](./imgs/warehouse.png)

### 금속 소재 적재

| 수량 | 적재 위치 |
|---|---|
| 1번째 금속 | 1 |
| 2번째 금속 | 3 |
| 3번째 금속 | 5 |

### 비금속 소재 적재

| 수량 | 적재 위치 |
|---|---|
| 1번째 비금속 | 2 |
| 2번째 비금속 | 4 |
| 3번째 비금속 | 6 |

각 소재는 종류별로 최대 3개까지 적재할 수 있습니다.

4번째 소재부터는 창고에 적재하지 않고 배출 동작을 수행합니다.

- 4번째 금속 소재: 배출 실린더를 통해 배출 창고로 이동
- 4번째 비금속 소재: 컨베이어 끝에 위치한 창고로 이동

적재 한도 이후 기존 적재 시퀀스가 실행되지 않도록 `C50`, `C51`의 B접점을 사용했습니다. 따라서 4번째 소재부터는 소재를 판별할 경우 C50, C51 카운트가 ON이 된 상태이고 기존 시퀀스가 끊어지도록 하였습니다.

## 5. 서보모터 위치 제어

서보모터는 QD77MS2 위치 결정 모듈을 통해 제어했습니다.

![서보모터 높이 위치 제어](./imgs/height.png)

![서보모터 위치 파라미터](./imgs/height_param.png)

서보모터가 이동하는 주요 위치는 다음과 같습니다.

- 원점 위치
- 소재 픽업 위치
- 1층 창고 전진 높이
- 2층 창고 전진 높이
- 3층 창고 전진 높이
- 소재를 놓기 위한 미세 하강 위치
- 흡착 실린더 후진을 위한 미세 상승 위치

소재를 잡거나 놓는 과정에서는 INC 위치 이동을 사용했습니다.  
이를 통해 각 층별 절대 위치 데이터 하나만 있어도, 해당 위치에서 약간 하강하거나 상승하는 동작을 구현할 수 있었습니다.

## 8. 전체 비움 모드

![전체 비움 순서 로직](./imgs/whole_out_flow.png)

전체 비움 모드는 창고에 적재된 6개의 소재를 정해진 순서대로 모두 배출하는 기능입니다.

### 전체 비움 순서

1. 1층 금속 위치
2. 2층 금속 위치
3. 3층 금속 위치
4. 1층 비금속 위치
5. 2층 비금속 위치
6. 3층 비금속 위치

![전체 비움 동작 1](./imgs/whole_out1.png)

전체 비움은 `M1008` 버튼으로 시작합니다.

![서보모터](./imgs/servomotor.png)

## 14. Tact Time 측정

작업 시간을 확인하기 위해 Tact Time 측정 기능을 추가했습니다.

![tacttime](./imgs/tacttime.png)

초기에는 PLC 내부 시계 데이터를 이용하는 방법을 검토했지만, 시간 값이 안정적으로 변환되지 않는 문제가 있었습니다.  
최종적으로는 1초 클록 특수 메모리인 `SM412`를 사용했습니다.

### Tact Time 동작 방식

1. 작업 시작 버튼 `X16` 입력
2. Tact Time 측정 시작
3. `SM412`가 1초마다 ON
4. 1초마다 `D412`에 1씩 누적
5. HMI에서 `D412`를 작업 경과 시간으로 표시
6. 작업 종료 후 측정 시간 유지
7. 다음 작업 시작 시 `D412` 초기화 후 재측정

## 15. 트러블슈팅

## 15.1 Tact Time 측정 문제

![트러블슈팅1](./imgs/trouble_tacttime.png)

### 문제

처음에는 PLC 내부 시계 데이터를 이용하여 작업 시작 시간과 종료 시간의 차이를 계산하려고 했습니다.  
하지만 시간 데이터가 실제 작업 시간과 맞지 않는 문제가 발생했습니다.

예를 들어 실제 동작 점검은 약 13초가 걸렸지만, BCD/16진수 변환 결과는 실제 시간과 맞지 않았습니다.

### 원인

PLC 내부 시계 데이터는 짧은 작업 시간 측정에 직접 사용하기 불편했고, 일부 값이 섞이거나 불안정하게 저장되는 문제가 있었습니다.

### 해결 방법

측정 방식을 변경했습니다.

- 내부 시계 데이터 사용 중단
- `SM412` 1초 클록 사용
- `D412`에 초 단위 누적
- HMI에서 `D412` 값을 Tact Time으로 표시

## 15.2 소재 수량에 따른 서보 위치 이동 오류

![트러블슈팅2](./imgs/trouble_2.png)

### 문제

`C50`, `C51` 카운터 값을 이용해 서보 위치를 선택하는 과정에서 서보모터가 잘못된 위치로 이동하는 문제가 발생했습니다.

예시 상황:

- 금속 3번째 소재를 적재해야 하는 상황
- 이미 비금속 1개가 적재된 상태
- `C51 = K1` 조건이 만족됨
- PLC 스캔 순서에 의해 잘못된 조건이 먼저 실행됨
- 서보모터가 의도하지 않은 위치로 이동

### 원인

기존 로직은 카운터 값만 확인했습니다.  
현재 적재하려는 소재가 금속인지 비금속인지 함께 확인하지 않았습니다.

PLC는 래더 프로그램을 위에서 아래로 스캔하기 때문에, 다른 소재 조건이 먼저 만족되면 잘못된 위치 이동이 발생할 수 있습니다.

### 해결 방법

위치 이동 조건에 현재 소재 판별 메모리를 추가했습니다.

- 금속 위치 이동 조건에 `M50` 추가
- 비금속 위치 이동 조건에 `M51` 추가

즉, 다음 두 조건을 동시에 만족할 때만 서보모터가 해당 위치로 이동하도록 수정했습니다.

```text
현재 소재 종류 조건
+
현재 소재 수량 조건
```

## 16. 작업 유의 사항

기본 자동 작업은 소재 종류와 수량에 따라 적재 위치가 자동으로 정해집니다.

따라서 선택 적재 후 바로 기본 자동 작업을 실행하면 같은 위치에 두 개의 소재를 적재하려고 할 수 있습니다.

### 예시

1. 비금속 소재를 선택 적재로 1층 금속 위치에 적재
2. 이후 기본 자동 작업 실행
3. 첫 번째 금속 소재가 1층 금속 위치로 이동
4. 이미 비금속 소재가 있는 위치에 다시 소재를 적재하려고 함
5. 창고 충돌 또는 장비 전도 위험 발생

따라서 다음 사항을 지켜야 합니다.

- 선택 적재 후에는 다음 작업도 선택 적재로 진행
- 선택 비움 후에도 이후 작업은 선택 작업으로 주의해서 진행
- 자동 작업 전 HMI의 창고 적재 상태 램프 확인

## 17. 유지보수 매뉴얼 요약

이 시스템은 소재를 금속 또는 비금속으로 판별하고, 판별 결과에 따라 지정된 창고 위치에 자동으로 적재하는 자동화 시스템입니다.

추가적으로 다음 기능을 제공합니다.

- 수동 JOG 조작
- 서보 위치 Teaching
- 가공 시간 설정
- 가공 반복 횟수 설정
- 소재 재투입 대기 시간 설정
- HMI 모니터링
- 선택 적재
- 선택 비움
- 전체 비움
- 비상 정지 및 비상 해제

전원이 공급되면 서보모터가 자동으로 원점 복귀하고 전체 실린더가 초기 위치로 이동합니다.  
따라서 전원을 공급하기 전 장비 주변을 정리해야 합니다.

시스템은 다음 조건을 만족할 때 작업 가능 상태가 됩니다.

- 서보 원점 확립
- 전체 실린더 후진
- 소재 공급 감지

HMI에서는 서보 위치, 서보 속도, 경고 번호, 에러 번호, 소재 판별 상태, 소재 수량, 작업 상태 메시지, Tact Time 등을 실시간으로 확인할 수 있습니다.
